TIOJ 2242

Created2022-06-04Updated2022-06-05

連結

題意

給定一棵 nnn 個點的樹，樹上每個節點會有一個數字 ava_vav​ 代表該節點的存水量
有 qqq 筆操作，每筆操作給定一個節點 vvv，請輸出點 vvv 當前的存水量，並清空點 vvv 的水
此外，其他點的水會往點 vvv 的方向移動一格

subtask

  1. subtask1：n,q≤5000n, q \leq 5000n,q≤5000, Score: 7
  2. subtask2：保證樹隨機生成，且每筆操作的答案皆非零, Score: 12
  3. subtask3：樹是一條鏈, Score: 22
  4. subtask4：無其他限制, Score: 59

對於 subtask 2, 3, 4，n,q≤105n, q \leq 10^5n,q≤105，對於所有測資，0≤av≤1090 \leq a_v \leq 10^90≤av​≤109
