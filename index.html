CF 1479D (樹壓平 + 持久化)

Created2021-02-14Updated2021-02-14

連結

題意

給一棵nnn個點的樹，點iii上有顏色aia_iai​
給定qqq筆詢問，每筆詢問有u,v,l,ru, v, l, ru,v,l,r，請輸出其中一個符合以下條件的顏色ccc：

  1. l≤c≤rl \leq c \leq rl≤c≤r
  2. 顏色ccc在uuu到vvv的路徑上出現奇數次
    若無滿足的顏色輸出-1

解法

首先要在樹上套資結可能會有點麻煩，先來考慮問題是序列的情況吧
做法可以先建持久化線段樹，留下每一個前綴的線段樹版本
這樣一來問題就會變成給定lll，rrr，要如何找出一個位置介於lll和rrr之間且數值相異
至於這要怎麼做呢？

這裡我是用hash，不確定有沒有其他較好的做法就是了
對於每一個線段樹節點用一個hash值把它對應區間的值串起來
這樣就相當單純了，只要判斷區間的hash值是不是相同，若相同就回傳-1
否則看左子樹和右子樹誰的hash值不同，就往那裏遞迴下去，跑到根就結束
複雜度會是好好的O(logn)\mathcal{O}(logn)O(logn)，因為線段數最多分成lognlognlogn個區間，又最多只會遞迴到根一次

解決完序列，接下來解決樹吧，可以直接建出樹壓平序列
這樣一來兩點的路徑會變成樹壓平序列的一段區間 + lca
又因為這一題的操作是xor，所以完全不用考慮序列重複元素的情況，寫起來會漂亮許多
樹壓平這裡就不細講，詳細可以看板中講義，那裡有蠻詳細的介紹

然後這題還有樹上莫隊 + 值域分塊的作法~

第一次寫靜置記憶體的持久化線段樹
我用的hash是，每個節點會生出兩個亂數x,yx, yx,y，那hash = (左兒子hash * x + 右兒子hash * y) % mod
然後還是被卡碰撞QQ，最後開了兩個hash才過

線性基

線性基通常被用來處理奇怪xor的問題
線性基BBB為一個整數集合AAA的子集，需滿足以下條件：

  1. 不存在一個subset的xor值為0
  2. 能由AAA的subset算出來的xor值，也能透過BBB的某個subset算出來

若我們把一個數二進位分解後，將他視為一個向量
像是4=(0,0,1,0,0),23=(1,0,1,1,1)4 = (0, 0, 1, 0, 0), 23 = (1, 0, 1, 1, 1)4=(0,0,1,0,0),23=(1,0,1,1,1)
用線代的觀點來看就是「任兩向量線性獨立」和「BBB可以張成AAA」
