題意

給定一個長度nnn的陣列，有qqq筆詢問，每筆詢問給定l,r,vl, r, vl,r,v
請找出l≤k<rl \leq k < rl≤k<r，區間[k,r)[k, r)[k,r)和vvv的xor最大值

1≤n≤104,1≤q≤106,0≤C<2301 \leq n \leq 10^4, 1 \leq q \leq 10^6, 0 \leq C < 2^{30}1≤n≤104,1≤q≤106,0≤C<230

解法

首先，假設算好suf[i]suf[i]suf[i]代表後綴[i,n)[i, n)[i,n)的xor值
由xor的性質推出答案 = suf[k]⊕suf[r]⊕vsuf[k] \oplus suf[r] \oplus vsuf[k]⊕suf[r]⊕v
那問題就回到要如何找出最好的kkk使那串數最大

再來思考一個較單純的問題，給定一個陣列a，有qqq筆詢問
每筆詢問給定一個數xxx，求ai⊕xa_i \oplus xai​⊕x的最大值
作法就是蓋一棵01-trie，先把陣列所有數字丟進trie裡
由於二進位的性質可以讓我們greedy取，假如最大位能夠出現1，就讓trie往那裡走

接下來要如何處理區間的問題
考慮一下持久化吧
存下每一個後綴的01-trie版本，並維護size
這樣在詢問的時候就能夠把兩棵trie取出來，利用size決定能不能往下走
最後決定出最大的答案

複雜度：O(qlogC)\mathcal{O}(qlogC)O(qlogC)
