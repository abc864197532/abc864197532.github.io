線性基

線性基通常被用來處理奇怪xor的問題
線性基BBB為一個整數集合AAA的子集，需滿足以下條件：

  1. 不存在一個subset的xor值為0
  2. 能由AAA的subset算出來的xor值，也能透過BBB的某個subset算出來

若我們把一個數二進位分解後，將他視為一個向量
像是4=(0,0,1,0,0),23=(1,0,1,1,1)4 = (0, 0, 1, 0, 0), 23 = (1, 0, 1, 1, 1)4=(0,0,1,0,0),23=(1,0,1,1,1)
用線代的觀點來看就是「任兩向量線性獨立」和「BBB可以張成AAA」
